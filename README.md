Tema escolhido: 
API de Gerenciamento de Leitura (Livros)

Descrição inicial:
O sistema consiste em uma API desenvolvida em Python que permite aos usuários gerenciar sua lista de livros, organizando quais já foram lidos, quais estão em andamento e quais desejam ler futuramente.

A API será utilizada por leitores que desejam manter um controle pessoal de suas leituras de forma simples e organizada.

O sistema também contará com uma organização no estilo kanban, onde os livros serão distribuídos em colunas como "Quero Ler", "Lendo" e "Lido", permitindo uma visualização clara do progresso de leitura.

O principal objetivo do sistema é ajudar na organização e acompanhamento da leitura, evitando esquecimentos e facilitando o planejamento de novos livros a serem lidos.
